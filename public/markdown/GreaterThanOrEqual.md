左辺が右辺より大きいか等しいかをチェックします。

```typescript
console.log(2 >= 4) // false
console.log(1.5 >= -1) // true
console.log(100 >= 100) // false
console.log('a' >= 'b') // false
console.log('い' >= 'あ') // true
```
